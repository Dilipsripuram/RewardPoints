# RewardPoints
 Spring Boot app that will return total reward points of a customer and show the results month by month
 
 
# Technical Details
---------------------------------------------------------------------------------------------------------------
- Spring Boot
- Maven
- H2
--------------------------------------------------------------------------------------------------------------
